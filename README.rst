===============
object_detector
===============

Lightweight object detector using pytorch lightning

Usage
=====

Installation
------------
.. code-block:: bash

  $ conda env create --file environment.yaml
  $ conda activate object_detector

Tests
-----
.. code-block:: bash

  $ pytest tests

Command Line Interface
----------------------
.. code-block:: bash

  $ object_detector --help
